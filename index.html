DICOPETS - PRIVACY POLICY

This Privacy Policy may change over time or be updated from time to time so we encourage and emphasize that you should continually review this page to ensure that you are happy with any such changes.

Information that is collected:
Dicopets does not collect any personal information (name, email, demographic information, etc).

Application permissions:
Dicopets uses the following sensitive application permissions:
- Access to the device's storage, including all its files: this permission is only used to load and save backup copies of the game data, and to store game screenshots (both actions are entirely optional). Files' content is never accessed other than for this purpose.
